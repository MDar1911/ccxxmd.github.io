基础入门之进入Unity的世界——笔记

06. Unity中的五种视图

Project =>工程视图
Scene / Game => 场景视图/游戏视图
Hierarchy / Inspector => 层级视图/检视视图(属性视图)

07. Unity中资源的分类

1、通过Unity创建的资源
2、外部导入的资源
3、通过Assets Store购买的资源

08. Unity中的游戏物体(Game Object)

场景：场景就是一个游戏环境
场景由游戏物体组成，在Hierarchy中所有的东西都是游戏物体

09. Unity中的组件Component

游戏物体由组件组成
Inspector的每一项都是组件

10. Unity中的场景

使用游戏场景可以分开不同的游戏关卡
减少一个游戏场景的载入时间
可以对不同关卡进行单独测试
注意：只能同时修改一个场景，当前项目中只有一个正在修改的场景
在项目运行情况下所做的修改都是临时的，是不可保存的

11.场景中的控制工具(快捷键)

双击聚焦一个游戏物体(聚焦:将scene的视野焦点移动到这个游戏物体的动作叫聚焦)
以下快捷键对应transform中的特定属性
对应右上角Hierarchy的四个键
Q:移动 W:位移 E:旋转 R:缩放

13. 场景视图中的操作

Iso视图下(平行视图下)
滚轮:放大缩小视图
鼠标右键移动:旋转视野(围绕焦点旋转视野)
鼠标左键:选择Game Objec游戏物体
鼠标中键按下移动:平移游戏场景视野

Persp视图下(透视视图下)
滚轮:放大缩小视图
鼠标右键移动:旋转视野(围绕当前视野位置旋转视野)
鼠标左键:选择Game Objec游戏物体
鼠标中键按下移动:平移游戏场景视野
Alt+鼠标左键:旋转视野(围绕焦点旋转视野)

14. 向量

坐标书写方式
一般坐标和向量的书写方式为 a , b , c按照 x , y , z 的顺序
在unity中我们使用x代表左右方向，一般右边为正方向；用y代表上下，用z代表前后
+x表示右边 +y代表上边 +z代表前面
